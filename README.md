xbdm
====

Xbox Debug Monitor client for Rust

## Usage

First, add the following to your `Cargo.toml`:

```toml
[dependencies]
xbdm = "0.1.0-alpha"
```

Next, add this to your crate root:

```rust
extern crate xbdm;
```
